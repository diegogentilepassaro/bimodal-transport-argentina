# Raw Data: Industrial Census

Argentina National Industrial Censuses, district level.

- Source: INDEC
- Years: 1954, 1985
- Variables: establishments, employment, wages, gross output, value added

## Citation

INDEC. Censo Nacional Económico 1954: Industria. Buenos Aires: INDEC, 1954.
INDEC. Censo Nacional Económico 1985: Industria. Buenos Aires: INDEC, 1985.
