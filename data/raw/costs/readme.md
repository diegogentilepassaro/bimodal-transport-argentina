# Raw Data: Transport Cost Parameters

Unit transport costs per ton-km by mode and cargo density sector.

- Source: Baumgartner & Palazzo (1969), Table II
- Used in: config.R (cost parameter block); consumed by
  code/pipeline/01_cost_raster.R and 03a_build_cost_raster.R
- See config.R for exact parameter values and cross-reference table

## Citation

Baumgartner, Jean-Pierre, and Pascual Santiago Palazzo. 1969.
"Estructura económica del transporte de carga automotor y ferroviario
en la Argentina." *El Trimestre Económico* 36 (143): 381–394.
https://www.jstor.org/stable/20856082

Author names and article metadata verified 2026-07-16 against the
publisher's own archive (eltrimestreeconomico.com.mx search listing:
"Jean-Pierre Baumgartner, Pascual Santiago Palazzo") and the JSTOR
record (vol. 36, no. 143(3), Julio-Septiembre 1969, pp. 381-394).

Note: the coauthor planning document cites this work as "Baumgartner,
T. and Palazzo, J. A. (1969), Consejo Nacional de Desarrollo, Buenos
Aires." Those initials do not match the journal publication. A CONADE
report version may exist as a separate item; if the cost parameters
were digitised from that report rather than the journal article, the
citation should be revisited (flagged for Cote).
