# Raw Data: Larkin Plan Segments

Digitized segment-level data from the 1962 Larkin Plan.
Key variable: studied (1) vs. not studied (0) — basis for main instrument.

- Source: Larkin, Thomas B. 1962. "Plan de largo alcance para los
  ferrocarriles argentinos." Buenos Aires: USAID/Ferrocarriles Argentinos.
- Accessed at: Mariano Moreno National Library, Buenos Aires
