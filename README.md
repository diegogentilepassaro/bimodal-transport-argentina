# Replication Package

AEA-compliant replication package for "Bimodal Transport Infrastructure
and Regional Development: Evidence from Argentina, 1960-1991."

This package is under construction. When complete, running `Rscript code/main.R`
will reproduce all tables, figures, and in-text numbers from raw data.
